Custom Types#

Vector#

Provide AbiVec<T,N>
status: valid
security: NO
safety: QM
reqtype: Functional
satisfies: stkh_req__communication__abi_compatible

An ABI-compatible AbiVec<T, N> type shall be provided in both C++ and Rust with the specified layout.

#[repr(C)]
pub struct AbiVec<T> {
   len: u32,
   capacity: u32,
   elements: [T; N],
}

template<typename T, std::size_t N>
struct AbiVec {
private:
   std::uint32_t len;
   std::uint32_t capacity;
   T elements[N];
};

Result#

Provide AbiResult<T,E>
status: valid
security: NO
safety: QM
reqtype: Functional
satisfies: stkh_req__communication__abi_compatible

An ABI-compatible AbiResult<T, E> type shall be provided in both C++ and Rust with the specified layout.

#[repr(C)]
pub struct AbiResult<T, E> {
   is_ok: u8,
   value: AbiResultUnion<T, E>,
}

#[repr(C)]
union AbiResultUnion<T, E> {
   ok: T,
   err: E,
}

template<typename T, typename E>
struct AbiResult {
private:
   std::uint8_t is_ok;
   union {
      T ok;
      E err;
   } value;
};
