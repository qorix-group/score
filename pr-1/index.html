Eclipse S-CORE

Welcome & Overview#

Welcome to the Software documentation of the S-CORE project.

Introduction#

Scope

The project develops an open source core stack for Software Defined Vehicles (SDVs), targeting embedded high-performance Electronic Control Units (ECUs). Check our Scope for details.

Project Status

Check out our roadmap to see the project’s key milestones and features we have planned. Click here to explore!

How to Contribute?

The only way to influence Eclipse S-CORE is to contribute - S-CORE is open! Please check our Contribute guide for more information.
